전체 명령어.

모든 명령은 단발 실행과 REPL 안에서 동일하게 동작합니다.
각 명령 뒤에 --help 를 붙이면 전체 옵션이 나옵니다.

market 시세

price현재가 — 여러 종목, --watch 폴링
overview종목 대시보드 — 시세·보유·차트·호가 (REPL: w)
watchlist관심종목 시세판 — 등락순·실시간 (REPL: wl)
orderbook호가창
trades최근 체결 내역
candles캔들 데이터 (1m | 1d)
chart터미널 캔들 차트 — 추세 확인
limits상한가 · 하한가

order 주문

buy매수 — 지정가 · 시장가 · 금액 기반(US)
sell매도
list주문 목록 — --all 전체 페이지
get주문 단건 조회
modify정정
cancel취소
commissions매매 수수료

account 계좌 · 자산

list계좌 목록 (accountSeq 확인)
holdings보유 주식 — 손익 · 수익률
buying-power매수 가능 금액 (KRW | USD)
sellable매도 가능 수량

stock 종목 정보

info기본 정보 — 시장 · ISIN · 상장일
warnings매수 유의사항

info 시장 정보

fx환율
calendar거래 캘린더 (KR | US)

auth 인증

status설정 확인 — 시크릿 마스킹
login토큰 강제 발급
logout토큰 캐시 삭제

전역 어디서나

repl대화형 셸 시작 (인자 없이 실행해도 동일)
-a, --account계좌 지정
--json원본 JSON 출력
--sim시뮬레이션 모드
--dry-run주문 전송 없이 미리보기 (buy · sell)

실거래 도구의
안전장치.

진짜 계좌에 주문이 나가는 도구이기에,
빠름보다 확실함이 먼저입니다.

--dry-run

전송 없는 미리보기

주문 요청 본문을 만들어 보여주기만 하고 전송하지 않습니다. 가격·수량·규칙 검증을 미리 통과시켜 보세요.

y/N

기본 확인 프롬프트

buy · sell · modify · cancel 은 기본적으로 확인을 거칩니다. 1억원 이상 주문은 별도 플래그까지 요구합니다.

--sim

격리된 모의 환경

시뮬레이션 상태는 로컬 파일에만 존재합니다. 실 계좌·실 API 와 완전히 분리된 채로 전체 흐름을 연습하세요.

스펙 검증

잘못된 주문은 떠나지 않습니다

지정가 가격 필수, 금액 주문은 미국 주식 전용 같은 규칙을 서버에 가기 전에 잡아냅니다.

지금, 터미널에서.

$ uv tool install tossinvest-cli

비공식 도구입니다. 토스증권과 무관하며, 사용에 따른 거래 결과의 책임은 사용자에게 있습니다. 실제 계좌에 주문이 전송되므로 --dry-run 과 시뮬레이션 모드로 충분히 연습한 뒤 사용하세요. API 사양은 공식 문서를 따릅니다.
